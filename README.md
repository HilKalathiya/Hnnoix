# Duranta Open5GS GUI 📡

Next-generation 5G network orchestration dashboard for real-time monitoring, configuration, and telemetry of the Open5GS core and gNB.

## ✨ Features
- **Live Network Topology**: Visual map of your Open5GS core, gNB, and UEs.
- **Real-time Log Streaming**: Built-in high-performance terminal to tail logs across all network functions with intelligent filtering.
- **Dynamic Configuration**: Elegant, easy-to-use interface for updating subscriber credentials (SIM), PLMN, and RAN parameters.
- **Modern UI**: Ultra-premium dark mode interface built with React, Tailwind CSS, and Lucide Icons.

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 🛠️ Tech Stack
- **Frontend**: React 18, Vite
- **Styling**: Tailwind CSS, Plus Jakarta Sans, JetBrains Mono
- **Icons**: Lucide React
- **Routing**: React Router
